# Location Unknown — HONNE

A modern single-page fan tribute website inspired by the song **"Location Unknown"** by HONNE ft. Georgia.

## ✨ Features

- **Dark space background** with animated stars and floating particles
- **Neural network aesthetic** — glowing nodes and connecting lines
- **Music player** with progress bar, volume control, and play/pause
- **Synced lyrics** that highlight as the song plays
- **Cinematic intro** overlay: *"Your location is unknown…"*
- **Subtle parallax** effect on mouse movement
- **Responsive design** — works on desktop and mobile

## 🚀 Deployment (cPanel / Static Hosting)

1. Upload these files to your `public_html` directory:
   - `index.html`
   - `style.css`
   - `script.js`
2. *(Optional)* Add a file named `song.mp3` in the same directory to enable music playback.
3. Open your domain in a browser — done!

No build step, no Node.js, no frameworks required.

## 🛠 Tech Stack

- **HTML5** — semantic structure
- **CSS3** — animations, gradients, backdrop blur, responsive layout
- **Vanilla JavaScript** — canvas animations, audio API, intersection observer
